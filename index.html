The UMS feature list.

UMS 功能列表(UMS feature list):


1. 验证码（图片，短信）校验功能(validate code (image, SMS) verification function)
2. 手机登录功能，登录后自动注册(Mobile login function, automatic registration after login)
3. 第三方登录功能(qq,weibo,weixin,gitee)，登录后自动注册，
　与用户账号绑定与解绑(OAuth2, binding and unbinding)
4. 登录路由功能 (login routing)
5. 统一回调地址路由功能(Unified callback address routing function)
6. 访问权限控制功能(Access control function)
7. 简化 session、remember me、crsf 等配置
　(Simplify session、remember me、srsf etc configuration)
8. 根据设置的返回方式（JSON 与 REDIRECT）返回 json 或 html 数据
9. 签到功能(sign)
